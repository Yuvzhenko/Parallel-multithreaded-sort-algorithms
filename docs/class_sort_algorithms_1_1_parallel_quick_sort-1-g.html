Detailed Description

Реалізація паралельного алгоритму швидкого сортування за допомогою Task Parallel Library (TPL).

Використовує обмеження глибини рекурсії (_maxDepth), щоб уникнути створення занадто великої кількості потоків для дрібних підмасивів.
